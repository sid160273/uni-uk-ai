# Uni-UK.AI

An AI-powered university discovery platform helping students find the perfect UK university based on their interests, goals, and preferences.

## Features

- Interactive AI chat assistant for personalized university recommendations
- Comprehensive database of 141 UK universities
- Detailed university profiles with rankings, facilities, and student life information
- Smart search with fuzzy matching
- Responsive design with modern UI

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript 5
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Search:** Fuse.js

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build for Production

```bash
npm run build
npm run start
```
